# Catalog API

A data catalog service for awesome data.

# Run

Python 3.5+

## Prepare database

```bash
CREATE DATABASE IF NOT EXISTS catalogdb DEFAULT CHARSET utf8 COLLATE utf8_general_ci;
```

List integrated commands:

```bash
invoke --list
```

Start app with shortcut:

```bash
invoke app.run
```
